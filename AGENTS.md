# AGENTS.md

本项目是“华悦快时尚服饰有限公司——零售门店数字化管理系统”。

请严格按照 README.md 中的需求、数据库设计、API 设计和 UI 页面要求开发。

技术栈：

- 后端：FastAPI + SQLAlchemy + SQLite
- 前端：React + Vite + TypeScript + Ant Design + ECharts

开发要求：

1. 优先实现可运行 MVP。
2. 不要一次性实现所有功能。
3. 每次只完成一个阶段。
4. 修改代码后说明改了哪些文件。
5. 后端必须能运行 http://localhost:8000/docs。
6. 前端必须能运行 http://localhost:5173。
7. 页面全部使用中文。
8. 不实现真实支付、真实短信、复杂权限和生产部署。
9. 不要删除 README.md 中已有需求。

## Project Context

本项目统一名称为“华悦快时尚服饰有限公司 —— 零售门店数字化管理系统”。

本项目是信息系统分析与设计课程项目，用于展示快时尚服装企业在门店收银、订单、库存、会员、促销和经营看板方面的数字化管理能力。

## Current Progress

当前已经彻底完成八个阶段中的第三阶段任务。第三阶段中的功能检查、问题定位和对应修复已经完成。

## Development Rules

- 修改代码前先阅读 README_DEV_STATE.md
- 修改前先说明计划和预计修改文件
- 不要大改项目架构
- 不要删除已有页面和已有功能
- 优先使用现有组件、mock data 和页面结构完成需求
- 每次修改后说明修改了哪些文件、如何运行、如何验证
- 如果发现需求与现有代码不一致，先询问再修改
